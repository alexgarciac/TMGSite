I

  • Illustrated books :
  • Illustration : One of several types of graphic elements used to demonstrate a point visually within a book. The term may also be used loosely to describe all of the graphic elements within a volume.
  • Illustration list : A list of all of the titles of the graphic elements within a book, usually found within the frontmatter and typically citing the page number where each can be found.
  • Illustrator :
  • Image : A type of graphic element found in a book that makes or describes a point visually. Images are typically created by means of a photographic process, and include reference to xrays, CT scans, and other radiographic views.

  • Independent Bookstore : Booksellers not affiliated with a major chain.
  • Indexing : Identification and structuring of the key concepts in a book, linking them to the specific pages on which they are found. Traditionally indexing is accomplished after the typeset pages have stabilized where there is little chance of reflow - generally after the first pass through composition. With XML it is possible to link concepts directly to their appearance in the manuscript via internal references that allow dynamic index creation without concern for reflow.
  • Index [Book] : An alphabetized list of important terms and concepts in a book accompanied by the page reference of each.
